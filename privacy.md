# Privacy Policy

**Last updated: April 26, 2026**

## 1. Overview
This application ("motivationpj") is an automated content pipeline operated by a single account owner. It does not collect, store, or share personal data from any third-party users.

## 2. Data Collected
- No personal data is collected from users.
- API tokens and credentials are stored securely and never shared.

## 3. Third-Party Services
This app uses the following third-party APIs:
- TikTok Content Posting API
- YouTube Data API v3
- Google Gemini API
- Pexels API

Each service has its own privacy policy. This app accesses only the minimum required scopes.

## 4. Data Retention
No user data is retained. API tokens are stored locally and in encrypted GitHub Secrets only.
